9.1 无线局域网WLAN

无线局域网：Wireless Local Area Network (WLAN)

9.1.1 无线局域网的组成

无线局域网可以分为两大类:

  1. 有固定基础设施的无线局域网：

    • 使用预先建立的基站覆盖固定地址
    • 采用 802.11 系列协议
    • 例如：蜂窝移动通信网使用电信公司的固定基站

  2. 无固定基础设施的无线局域网：

    • 移动自组网络
    • 例如：蓝牙

802.11协议相关概念：

  • 无线以太网标准是 802.11 系列协议
  • 使用 802.11 系列协议的局域网称为 Wi-Fi
  • WLAN 表示无线局域网，Wi-Fi 是采用 802.11 协议的局域网
  • Wi-Fi 已成为 WLAN 的代名词

802.11网络架构：

  • 使用星形拓扑
  • 中心是接入点 AP
  • MAC 层使用 CSMA/CA 协议和停止等待协议
  • 最小构件是基本服务集 BSS
    • 包括一个基站和若干移动站
    • 接入点 AP 就是 BSS 内的基站

BSS通信特点：

  • BSS 内站点可直接通信
  • 与其他站通信需通过接入点 AP
  • 每个 AP 有服务集标识符 SSID（即wifi名字）
  • BSS 覆盖范围称为基本服务区 BSA（直径≤100m）
  • AP 之间使用有线连接

移动站漫游过程：

  1. 选择接入点 AP 建立关联
  2. 通过 AP 发送 DHCP 发现报文获取 IP 地址
  3. 作为该 AP 子网成员加入网络

移动站连接特点：

  • 通常选择信号最强的 AP
  • AP 信道资源有限
  • 采用 802.11 协议通信
  • 现代设备内置无线网卡，实现 802.11 物理层和 MAC 层功能
  • 一般采用 WPA 或 WPA2 加密

移动自组网络：

  • 无固定基础设施的无线局域网
  • 无基站，移动站平等通信形成临时网络
  • 一般不与外界网络连接
  • 典型例子：蓝牙
  • 无线传感器网络 WSN 是快速发展的自组网络
    • 由大量传感器节点通过无线通信构成
    • 物联网 IoT 是其应用领域

9.1.2 802.11局域网的物理层

802.11网络基本架构：

  • 使用星形拓扑
  • 中心是接入点 AP
  • 接入点 AP 是基本服务集内的基站

WiFi版本发展：

  • **WiFi6 (IEEE 802.11ax)**：
    • 2020年6月正式发布
    • 又称 HEW（High Efficiency WLAN）
    • 支持 1GHz 到 6GHz 的所有 ISM 频段
    • 包括 6GHz、2.4GHz 和 5GHz
    • 向下兼容 a/b/g/n/ac
    • 最高带宽为 160MHz
    • 单条流最高速率 1201Mbit/s

历史版本：

  • 第 4 代 wifi：802.11n
  • 第 5 代 wifi：802.11ac

9.1.3 802.11局域网的MAC层协议

协议对比：

  • CSMA/CA：载波监听多点接入/碰撞避免
  • CSMA/CD：载波监听多点接入/碰撞检测

802.11 MAC层特点：

  • 使用 CSMA/CA 协议和停止等待协议
  • 使用停止等待原因：无线信道通信质量差于有线信道
  • 不能使用 CSMA/CD 原因：无线局域网中站点不能都听见对方

MAC层协议的协调功能：

  1. 分布协调功能（DCF）：

    • 不采用中心控制
    • 每个节点使用 CSMA 机制
    • 通过争用信道获取发送权

  2. 点协调功能（PCF）：

    • 可选功能
    • 用接入点 AP 集中控制
    • 采用轮询方式分配发送权
    • 适用于时间敏感业务

802.11采用的机制：

  1. 帧间间隔时间：

    • 完成发送后等待帧间间隔时间
    • 不同优先级帧具有不同帧间间隔 IFS
    • 可降低碰撞概率

  2. 虚拟载波监听：

    • 源站通知其他站占用时间
    • 其他站在此期间停止发送
    • 可降低碰撞概率

  3. 随机退避算法：

    • 信道空闲时等待 DIFS 间隔
    • 执行退避算法
    • 维护退避计时器
    • 计时器归零后立即发送

  4. 预约机制：

    • 发送数据前先发送 RTS 帧
    • 收到 CTS 帧后发送数据
    • 预约帧指明预约时间
    • 其他站在预约期间不发送
    • 可选择性使用

9.1.4 802.11局域网的MAC帧

MAC帧的基本特点：

  • 802.11 的 MAC 帧有三种类型：

    • 控制帧
    • 数据帧
    • 管理帧

  • MAC 帧首部有 30 字节

  • 尾部是 4 字节的帧检验序列

数据帧地址字段：

  • 802.11 数据帧有4个地址字段
  • 地址4用于自组网络
  • 前三个地址内容取决于帧控制字段中的”去往 AP“和”来自 AP“

9.2 无线个人区域网WPAN

WPAN特点：

  • 把个人设备用无线技术连接的自组网络
  • 工作在 2.4GHz 频段
  • 包括：
    • 蓝牙系统
    • ZigBee
    • 超高速 WPAN

9.3 无线城域网WMAN

WMAN特点：

  • 可提供最后一英里的宽带无线接入
  • 可代替现有的有线宽带接入
  • 标准是 802.16 系列协议
  • 可覆盖一个城市的部分区域

9.4 蜂窝移动通信网

9.4.1 蜂窝无线通信技术简介

移动通信类型：

  • 蜂窝移动通信
  • 卫星移动通信等

蜂窝移动通信特点：

  • 属于小区制移动通信
  • 网络划分为多个小区（蜂窝）
  • 每个小区设置一个基站
  • 移动站通信必须通过基站完成
  • 现代蜂窝移动通信网采用了许多 IP 技术

移动通信发展：

  • 2G（第二代）：

    • 带宽为 200kHz
    • 代表是 GSM 系统
    • GPRS 也是 2G 技术
    • 基本只能提供电话和短信服务

  • 3G：

    • 带宽为 5MHz
    • 使用 IP 体系结构
    • 采用混合交换体制（电路交换和分组交换）
    • 以传输业务为主的通信系统

3G/4G特点：

  • 诞生了上网卡
  • 上网卡可插入电脑 USB 接口
  • 通过 3G/4G 网络接入互联网
  • 用户共享带宽，实际分配带宽不确定

蜂窝结构特点：

  • 每个基站发射功率要能覆盖本小区
  • 避免干扰邻近小区
  • 采用蜂窝结构最大化频分复用
  • 每个基站使用不同频率

9.4.2 移动IP

基本概念：

  • 允许计算机移动到外地时仍保留原来的 IP 地址
  • 与 DHCP 的区别：
    • DHCP：换位置时自动获取新IP地址
    • 移动 IP：在移动中保持原IP地址

移动IP的重要概念：

  • 永久地址/归属地址
  • 归属网络
  • 被访网络
  • 归属代理
  • 外地代理
  • 转交地址
  • 同址转交地址

移动IP使用的协议：

  1. 移动站到外地代理的协议
  2. 外地代理到归属代理的登记协议
  3. 归属代理数据报封装协议
  4. 外地代理拆封协议

实现机制：

  • 基本要素：

    • 移动站必须有归属地址（永久地址）
    • 原始连接的网络为归属网络
    • 归属代理通常是归属网络上的路由器

  • 漫游过程：

    1. 移动站移动到外地网络（被访网络）
    2. 被访网络使用外地代理（通常是路由器）
    3. 外地代理负责：
      • 创建临时转交地址
      • 通知归属代理转交地址

通信过程示例：

  1. 通信者B发送数据到移动站A：

    • B发送数据报（目的地址为A的永久地址）
    • 归属代理截获数据报
    • 归属代理封装数据报（使用隧道技术）
    • 发送到外地代理（使用A的转交地址）
    • 外地代理拆封并转发给A

  2. 移动站A回复B：

    • 使用永久地址作为源地址
    • 直接发送到B的IP地址
    • 不需要经过归属代理

9.4.3 蜂窝移动通信网中对移动用户的路由选择

路由选择方式：

  • 间接路由选择
  • 直接路由选择（需要通信者代理和锚外地代理）

移动交换中心数据库：

  1. 归属位置寄存器 HLR：

    • 类似归属代理功能

  2. 来访用户位置寄存器 VLR：

    • 类似外地代理功能

9.4.5 无线网络对高层协议的影响

主要问题：

  • 移动站漫游时网络连接中断
  • TCP 报文段频繁丢失
  • TCP 拥塞控制受影响
  • 拥塞窗口被错误缩小

解决方法：

  1. 本地恢复
  2. 让 TCP 发送方知道无线链路位置
  3. 将含移动用户的端到端 TCP 连接拆分：
    • 移动用户到无线接入点一个 TCP 连接
    • 剩余有线网络另一个 TCP 连接

9.5 两种不同的无线上网

收费方式对比：

  1. 蜂窝移动网络：

    • 按流量计费

  2. WiFi：

    • 通过宽带上网
    • 根据带宽和使用时间收费
